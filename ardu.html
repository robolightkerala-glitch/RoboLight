Arduino Core Specifications

Microcontroller

ATmega328P 8-bit RISC architecture running at 16 MHz clock speed, delivering reliable deterministic execution for robotics tasks.

Operating Voltage

Standard 5V logic operating level with support for 7V–12V unregulated external DC input for direct battery pack power.

Memory Architecture

32 KB Flash Memory (with 0.5 KB used by bootloader), 2 KB SRAM, and 1 KB EEPROM for persistent setting storage.

Shield Ecosystem

Standard header pin footprint supporting thousands of plug-and-play expansion shields for motor drivers, GPS, and relays.

Arduino I/O & Sensor Header

Flexible Input/Output

Direct low-latency hardware control. Hook up ultrasonic sensors, line-following IR modules, and servo motors effortlessly.

  • 14 Digital I/O Pins: Precise digital read and write operations for switches, LEDs, and sensors.
  • 6 PWM Outputs: Hardware-based 8-bit pulse-width modulation for smooth motor speed control.
  • 6 Analog Inputs (ADC): 10-bit analog channels to read analog sensor voltages (0V–5V).
  • Dedicated Communication: Built-in UART Serial (TX/RX), SPI, and I2C (TWI) bus headers.
